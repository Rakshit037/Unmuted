# 🎤 Comedy Show Booking Platform

A full-stack MERN application for booking stand-up comedy shows with:

## 🚀 Features

* Role-based authentication (Admin & Users)
* Comedian management (CRUD)
* Show & slot management
* Dynamic seat selection (Balcony, Royal, etc.)
* Seat locking system
* Booking system
* Email notifications
* Reviews & ratings

## 🛠️ Tech Stack

* MongoDB (Atlas)
* Express.js
* React.js
* Node.js
* MUI (Frontend)
* JWT + HTTP-only cookies
* Nodemailer
